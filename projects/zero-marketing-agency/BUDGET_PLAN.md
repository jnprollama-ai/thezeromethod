# Zero Marketing Agency - Budget-Constrained Plan

## Budget: $50 Maximum (Prefer $0)

### Phase 1: Zero-Cost Foundation ($0)

**Stack (100% Free):**
- **Domain:** Freenom (.tk, .ml domains) OR subdomain (zeromarketing.netlify.app)
- **Hosting:** Netlify / Vercel / Cloudflare Pages (free tier)
- **CMS:** Markdown + Git (GitHub public repo)
- **Email:** ConvertKit free tier (up to 1,000 subscribers)
- **Payments:** Gumroad (no upfront, 5% fee)
- **SEO/Analytics:** Google Search Console + GA4 (free)
- **Images:** Unsplash / AI-generated (free)
- **Social:** Direct posting (no scheduler needed initially)

**Trade-offs:**
- Freenom domains: Less professional but free
- Subdomain: Looks less legit but totally functional
- No paid tools: More manual work for me, but doable

### Phase 2: Professional Upgrade ($30-50)

**If we hit $100 in revenue, upgrade:**
- Real domain (.com/.co) - $10-15/year
- ConvertKit paid ($0-29/month) - when we exceed free tier
- Buffer/ scheduling tool ($0-15/month) - if needed
- Total: ~$40-50 first month
